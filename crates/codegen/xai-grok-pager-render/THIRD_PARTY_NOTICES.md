## warpdotdev/themes

This crate embeds 340 YAML/YML theme definitions from
[`warpdotdev/themes`](https://github.com/warpdotdev/themes) at revision
`b385044250f1ed3c9379ab34a8fe82f02fdffaa4`. Preview SVGs and background image
files are excluded. The upstream Apache License 2.0 text is preserved at
`assets/warp-themes/LICENSE`. Binary NPM packages ship the same full text as
`WARP_THEMES_LICENSE` and include this notice in `THIRD_PARTY_NOTICES.md`.

Individual source comments identify these MIT-derived definitions and are
retained in the embedded YAML:

- `standard/cherry.yaml`, a port of the VS Code Cherry theme — Copyright (c)
  2020 Nauval Rizky.
- `standard/soft_tactile_warp_cream.yaml`,
  `standard/soft_tactile_warp_dark.yaml`, and
  `standard/soft_tactile_warp_light.yaml` — authors Jelena
  (<https://github.com/jelenastricak>) and Bob
  (<https://github.com/bobvasic>) at CyberLink Security.
